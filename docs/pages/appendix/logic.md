# 論理

## 論理範疇はプリャンナでどのように表現されるか？

**論理演算子（含意、同値）**は法の範疇では表現されず、**別の形態**で扱っている。法の範疇ではより純粋な**「様相」**と**「認識」**に特化して設計されている。

また、**様相論理**（義務論理、認識論理、時相論理）や**非古典論理**（直観主義、ファジィ）の概念も含まれる。

## 1. 論理的構造の表現手段

| 論理 | 法・形態 | スロット |
| :--- | :--- | :--- |
| **命題論理** (含意 $\rightarrow$, 同値 $\leftrightarrow$) | 法では扱わない | **意味派生 (Semantic Derivation, スロット 2)**: 語彙レベルで動詞を「関係動詞」化するのに最適。**接続詞**としても機能させる。 |
| **様相論理** (可能性 $\Diamond$, 必然性 $\Box$) | 必然法 ($\text{g}$), 可能性法 ($\text{h}$) | **法 (Mood, スロット 7)**: 話し手の命題への認識態度を表すため、最適。 |
| **義務論理** (義務 $\text{O}$, 許可 $\text{P}$) | 命令法 ($\text{t}$), 許容法 ($\text{z}$) | **法 (Mood, スロット 7)**: 強い働きかけと社会的な容認をカバーしているため、十分。 |
| **認識論理** (知っている $\text{K}$, 信じている $\text{B}$) | 蓋然法 ($\text{m}$) | **証拠性 (Evidentiality, スロット 11)**: 情報源と確信度を扱うため、最適。 |
| **時相論理** (過去のある時 $\text{P}$, 未来のある時 $\text{F}$) | 時制 ($\text{w}\text{V}$), 時間格類 | **時制 (Tense, スロット 6)** および **時間格**: 既に時間枠を完全にカバーしているため、法で扱う必要はない。 |

## 2. 非古典論理のカバー

### A. 直観主義論理 (Intuitionistic Logic)

古典論理と異なり、「**排中律** ($\text{P} \vee \neg \text{P}$) は成り立たない」とし、証明されたもののみを真と見做す論理。

* **カバー方法**: **証拠性 (Evidentiality, スロット 11)** を設計する際に、**「証明された証拠」**や**「構成的証明」**を示す証拠性マーカーを導入することで対応できる。

### B. ファジィ論理 (Fuzzy Logic)

真理値が $\{0, 1\}$ ではなく、連続的な $[0, 1]$ の範囲で表現される論理。

* **カバー方法**: **蓋然法 ($\text{m}$)** を拡張する。蓋然法自体は**高い確信度**を示すが、これを**限定格（スロット28）**や**数量詞**と組み合わせることで、**確率的な真理値（例: $0.7$ の真理値で〜だろう）**を表現できるように統語論を設計する。

## 3. 法 (Mood) の表しうる論理

| :--- | :--- | :--- | :--- | :--- |
| 1. | **直説法** | $\text{l}$ | **中立的な事実**（デフォルト） | 現実性 |
| 2. | **蓋然法** | $\text{m}$ | **内部認識**（高い確信の推測） | 認識論理 / ファジィ |
| 3. | **仮定法** | $\text{p}$ | **仮説の境界**（非現実の仮説） | 論理構造 |
| 4. | **命令法** | $\text{t}$ | **最も強い強制** | 義務論理 |
| 5. | **要請法** | $\text{s}$ | **丁重な依頼** | 義務論理 |
| 6. | **勧誘法** | $\text{k}$ | **共同行動の提案** | 義務論理 |
| 7. | **祈願法** | $\text{r}$ | **運命への期待**（強い願望） | 義務論理 |
| 8. | **許容法** | $\text{z}$ | **境界の緩和**（許容） | 義務論理 |

| 11. | **可能性法** | $\text{h}$ | **論理的な可能性** ($\Diamond$) | 様相論理 |
