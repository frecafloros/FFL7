# プリャンナ語の音素配列論 (Phonotactics)

ここでは、プリャンナ語において許容される音素の並び（音素配列）に関する規則の概要を記述します。
これにより、言語の音韻的な一貫性を保ち、発音可能な単語の構造を定義しています。

言語の音韻構造に関する厳密な定義は、以下のBNF（バッカス・ナウア記法）ファイルに自己完結した形で記述されています。言語の仕様に関する一次情報源として、また、言語処理ツールを開発する際の参照先として利用してください。

**正式な定義ファイル:** [`pryanna.bnf`]({{ "/appendix/pryanna.bnf" | relative_url }})

## 構造の基本方針

プリャンナ語の語根は、`CVC`（子音-母音-子音）構造を基本としています。
この構造と音素の組み合わせは、「緊張と緩和」の原則に基づいて設計されており、言語の個性と発音のしやすさを両立させることを目的としています。

### 1. 音素のグループ分け

音素は、その音韻的な特徴や発音の難易度に基づき、大きく3つの「緊張度」グループに分類されます。

*   **グループA (緩和音):** 最も基本的で、発音しやすい音。
*   **グループB (中間音):** やや複雑さを増し、言語に深みを与える音。
*   **グループC (緊張音):** プリャンナの個性を際立たせる特殊で発音が難しい音。

各グループに属する具体的な音素のリストは、上記の `pryanna.bnf` ファイル内で定義されています。

### 2. 語根の構造ルール

語根の構造は、以下のルールに従います。

1.  **語頭の子音 (C1):** 全てのグループ（A, B, C）の子音が来ることができます。
2.  **語末の子音 (C2):** 発音のしやすさを考慮し、**緩和子音（グループA）**のみが来ることができます。
3.  **母音 (V):** 全てのグループ（A, B, C）の母音が来ることができます。

### 3. 特別な制約ルール

上記の構造ルールに加え、以下の重要な制約が存在します。

*   **緊張の連続の回避 (Tension Avoidance):**
    *   もし語頭の子音(C1)が**緊張子音（グループC）**である場合、続く母音(V)は**緩和母音（グループA）**でなければなりません。
    *   これにより、`[緊張子音] + [緊張母音]` のような、発音が特に困難になる組み合わせが生成されるのを防ぎます。

この制約は、`pryanna.bnf` ファイル内ではコメントとして記述されており、直接表現していない。
