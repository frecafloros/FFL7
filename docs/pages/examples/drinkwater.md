# 語用例：「水を飲む」

## 基本形

>    
> 
> *h-a-pha tan-a-n tac-æf-yeũ-m-wa-l-odh-ã.*
> 
> *私(単数, 能格) 水(単数, 対格) 飲む(継続活動, 進行相, 能動, 直説法, 現在, 直接証拠, 肯定).*
>
> 「私が水を飲んでいるのだ。」

* 動詞：- *tac-*（飲む）
* 能格：- *h-*（私）
* 対格：- *tan-*（水）

### 1. 動詞：「飲む」(tac)
「（今まさに）飲んでいる最中だ」というニュアンスにすると、

* 語根 (*√tac*): tac
* 語彙アスペクト: 「継続活動」の -æf 
* 文法アスペクト: 「進行相」の -yeũ で「～している最中」感を出す
* 態: もちろん「能動態」の -m
* 時制: 「現在時制」の -wa で「今」を表現
* 法: 事実を述べる「直説法」の -l
* 証拠性: 自分で飲んでるから「直接証拠性」の -dh
* 極性: 肯定文だからã

全部合体させると…

`tacæfyeũmwalodh`

### 2. 能格：「私」(h)

「私が」という動作の主を表すために「能格」(-pha) をくっつける。

* 語根 (*√h*): h
* 数：「単数」の -a
* 格: 「能格」の -pha

合体させると

`hapha`

これで「私が」の意味になる。

### 3. 対格：「水」(tan)

「水を～される」という、動作の対象を表すために「対格」(-n) をくっつける！

* 語根 (*√tan*): tan
* 数：「単数」の -a
* 格: 「対格」の -n

合体させると

`tanan`

これで「水を」だ！
