# 格と態の連携

ここでは、動詞の語彙アスペクトは**基底状態**（$\text{-}\text{∅}$）、文法アスペクトは**中立相**（$\text{-}\text{∅}$）と仮定し、**態の接辞**と**統語論格の音素**のみに焦点を当てる。

## 統語論格（9個）と態（8個）の連携検証

特に複雑な**能格/自発格**、**刺激格/経験格**の対立、および**具格/手段格**の焦点化に注目する。

| 統語論格 (9個) | 態の選択と焦点化（主語の役割） | 例文の構造と検証 |
| :--- | :--- | :--- |
| **1. 能格** ($\text{ph}\text{a}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{Actor}_{\text{pha}} + \text{Verb}_{\text{m}} + \text{Object}_{\text{n}}$. (俳優が意図してドアを開けた。) **→** 能動態で能格が主語として標準的に機能。 |
| **2. 自発格** ($\text{tʃ}\text{hy}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{Wind}_{\text{tʃhy}} + \text{Verb}_{\text{m}} + \text{Object}_{\text{n}}$. (風が勝手にドアを閉めた。) **→** 能動態で自発格が主語となり、**非意図的行為**を表現。能格との役割分離を確認。 |
| **3. 刺激格** ($\text{l}\text{i}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{Noise}_{\text{li}} + \text{Verb}_{\text{m}} + \text{Experiencer}_{\text{sa}}$. (騒音が彼女に不安を感じさせた。) **→** 刺激格が能動態の主語となり、**経験格** ($\text{sa}$) の動作を引き起こす。 |
| | **受動態** ($\text{l}$) | $\text{Noise}_{\text{li}} + \text{Verb}_{\text{l}} + \text{Agent}_{\text{pha}}$. (騒音が俳優によって引き起こされた。) **→** 刺激格は**主語として維持**され、行為者 ($\text{pha}$) が非焦点化される。 |
| **4. 経験格** ($\text{s}\text{a}$) | **中動態** ($\text{n}$) | $\text{Child}_{\text{sa}} + \text{Verb}_{\text{n}} + \text{Stimulus}_{\text{li}}$. (子供が恐怖で震えた。) **→** 経験格が中動態の主語となり、**自己完結的な経験**を表現。 |
| | **受動態** ($\text{l}$) | $\text{Child}_{\text{sa}} + \text{Verb}_{\text{l}} + \text{Agent/Stimulus}$. (子供が外部の力によって感じさせられた。) **→** 主語として維持され、**経験の受け手**であることを強調。 |
| **5. 対格** ($\text{n}$) | **逆受態** ($\text{l}$) | $\text{Object}_{\text{n}} + \text{Verb}_{\text{l}} + \text{Actor}_{\text{pha}}$. (ボールが俳優によって打たれた。) **→** 受動態によって**状態変化の対象**である対格が主語となり、能格を非焦点化。 |
| **6. 題格** ($\text{r}\text{a}$) | **逆受態** ($\text{l}$) | $\text{Mountain}_{\text{ra}} + \text{Verb}_{\text{l}} + \text{Observer}_{\text{pha}}$. (山が観察者によって見られた。) **→** 受動態によって**状態変化のない対象**である題格が主語となる。対格との役割分離を確認。 |
| **7. 具格** ($\text{fw}\text{a}$) | **使役能動態** ($\text{f}$) | $\text{Axe}_{\text{fwa}} + \text{Verb}_{\text{f}} + \text{Object}_{\text{n}} + \text{Actor}_{\text{pha}}$. (斧が俳優によって木を切ることを使役した。) **→** 具格を能動態的に**「道具として作用する」**主語として機能させる（能格を非焦点化）。 |
| **8. 手段格** ($\text{v}$) | **使役中動態** ($\text{x}$) | $\text{Telepathy}_{\text{v}} + \text{Verb}_{\text{x}} + \text{Communicator}_{\text{pha}}$. (テレパシーが伝達者自身のために伝達することを仕向けた。) **→** 手段が**自己完結的に作用する**主語として機能。具格との役割分離を確認。 |
| **9. 因格** ($\text{i}$) | **原因焦点態** ($\text{m}$) | $\text{Traffic}_{\text{i}} + \text{Verb}_{\text{m}} + \text{Object}_{\text{n}}$. (渋滞が私たちを遅刻させた。) **→** 能動態で因格が主語となり、**行為の直接原因**として焦点化される。 |

特に、

* **能格 vs. 自発格**: 能動態（$\text{m}$）を使用することで、**意図的（能格）と非意図的（自発格）**な行為者を明確に分離した上で、両者を文の主語として扱える。
* **対格 vs. 題格**: 受動態（$\text{l}$）を使用することで、**状態変化の有無**にかかわらず、両者を行為の受け手（主語）として焦点化できる。
* **刺激格 vs. 経験格**: 能動態と中動態の使い分けにより、**原因（刺激格）**と**受け手（経験格）**の両方を主語として独立して表現できる。
* **具格 vs. 手段格**: 使役態と組み合わせることで、**物理的な道具**と**抽象的な方法**を「作用の主体」として焦点化できる。


## 関係格（11個）と態（8個）の連携検証

関係格は、動詞の引数（argument）間の抽象的、所有、または比較の関係を確立する。これらの格が、動詞の態とどのように連携し、それぞれの役割が明確に保たれているかを検証する。

ここでは、**与格、創出格、互格、基準格**など、行為や関係性に深く関わる格に焦点を当てる。

| 関係格 (11個) | 態の選択と連携（主語の役割） | 例文の構造と検証 |
| :--- | :--- | :--- |
| **1. 与格** ($\text{m}\text{i}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{I}_{\text{pha}} + \text{gave}_{\text{m}} + \text{book}_{\text{n}} + \text{to her}_{\text{mi}}$. (私が本を**彼女に**与えた。) **→** 与格は間接的な受け手として能動態で標準的に機能。主語（能格）とは明確に分離。 |
| | **使役能動態** ($\text{f}$) | $\text{I}_{\text{pha}} + \text{caused}_{\text{f}} + \text{her}_{\text{mi}} + \text{to receive}_{\text{m}} + \text{book}_{\text{n}}$. (私が**彼女に**本を受け取らせた。) **→** 与格が使役の対象として登場し、受動的な受益者としての役割を保つ。 |
| **2. 奪格** ($\text{n}\text{i}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{He}_{\text{pha}} + \text{fled}_{\text{m}} + \text{from the fear}_{\text{ni}}$. (彼が**恐怖から**逃れた。) **→** 奪格は非物理的な起源・分離点として機能。態による主語化は起こらない。 |
| **3. 属格** ($\text{l}\text{a}$) | **中動態** ($\text{n}$) | $\text{The ring}_{\text{n}} + \text{is made}_{\text{n}} + \text{of gold}_{\text{la}}$. (その指輪は**金で**作られている。) **→** 中動態（自動発生的影響）と結合し、**材質**という静的な属性（属格）が動作の内部的な性質として表現される。 |
| **4. 内属格** ($\text{h}\text{i}$) | **反射態** ($\text{y}$) | $\text{He}_{\text{pha}} + \text{hurt}_{\text{y}} + \text{his arm}_{\text{hi}}$. (彼は**彼の腕を**傷つけた。) **→** 反射態により、動作が主語自身に意図的に反射される際、内属格（体の部分）がその対象となる。 |
| **5. 外属格** ($\text{hw}\text{u}$) | **受動態** ($\text{l}$) | $\text{The car}_{\text{n}} + \text{was sold}_{\text{l}} + \text{by him}_{\text{hwu}}$. (その車は**彼によって**売られた。) **→** 外属格（譲渡可能な所有物）が受動態の文脈で所有者（譲渡元）をマークする。 |
| **6. 創出格** ($\text{v}'\text{o}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{This is}_{\text{m}} + \text{painting}_{\text{n}} + \text{by the artist}_{\text{v'o}}$. (これは**芸術家による**絵だ。) **→** 創出格は**動的な起源**として機能。動詞（is/be）が能動態であることで、創作という行為の結果であることを強調。 |
| **7. 分格** ($\text{r}\text{o}$) | **継続活動相** ($\text{-}\text{æf}$) | $\text{I}_{\text{pha}} + \text{drink}_{\text{m, -æf}} + \text{water}_{\text{ro}}$. (私が**水の一部を**飲み続けている。) **→** 分格は、語彙アスペクトが**継続活動**である動詞と結合し、全体から部分を継続的に消費する行為を表現。 |
| **8. 集合格** ($\text{ʑ}\text{e}$) | **中動態** ($\text{n}$) | $\text{He}_{\text{n}} + \text{is a member}_{\text{n}} + \text{of the team}_{\text{ʑe}}$. (彼は**そのチームの**一員だ。) **→** 中動態（自己に影響）が集合への**所属**という自己規定的な状態を表現。 |
| **9. 互格** ($\text{tʃ}'\text{e}$) | **中動態** ($\text{n}$) | $\text{They}_{\text{pha}} + \text{love}_{\text{n}} + \text{each other}_{\text{tʃ'e}}$. (彼らは**互いに**愛し合っている。) **→** 互格は、中動態と結合することで、**対等な対称性**を持つ自己完結的な相互作用を表現。 |
| **10. 基準格** ($\text{k}\text{æ}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{She}_{\text{n}} + \text{is faster}_{\text{m}} + \text{than him}_{\text{kæ}}$. (彼女は**彼よりも**速い。) **→** 比較の**基準点**として機能。態の変更は役割に影響しない。 |
| **11. 役割格** ($\text{p}\text{y}$) | **達成相** ($\text{-}\text{u}$) | $\text{The ice}_{\text{n}} + \text{turned}_{\text{-u}} + \text{into water}_{\text{py}}$. (氷が**水へと**変わった。) **→** 役割格は、文法アスペクトが**達成相**である動詞と結合し、**変化の結果**としての新しい状態を表現。 |

関係格の11個は、動詞の態（中動態、反射態、使役態）やアスペクトと連携することで、その**抽象的な関係性のニュアンス**を維持しつつ、文中で明確に機能する。特に、**所有の3分類**（内属、外属、創出）や、**相互作用の2分類**（互格、基準格）の分離が強固。

## 意味論格（7個）と態（8個）の連携検証

意味論格は、文の核となる構造（主語、目的語、主要な関係）から独立した、**付帯的な情報、様態、態度**などを表現する。これらの格が動詞の態とどのように連携し、意味論的な役割を純粋に果たしているかを検証する。

ここでは、**様態、欠如、同伴**など、動作の周縁的な情報に関わる格に焦点を当てる。

| 意味論格 (7個) | 態の選択と連携（主語の役割） | 例文の構造と検証 |
| :--- | :--- | :--- |
| **1. 呼格** ($\text{ʔ}\text{e}$) | **全ての態** ($\text{m}, \text{n}, \dots$) | $\text{Friend}_{\text{ʔe}}! + \text{You}_{\text{pha}} + \text{opened}_{\text{m}} + \text{the door}_{\text{n}}$. (**友よ**! 君がドアを開けた。) **→** 呼格は文法構造から完全に**独立**しており、動詞の態に影響を与えないことを確認。 |
| **2. 共格** ($\text{m}\text{u}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{I}_{\text{pha}} + \text{went}_{\text{m}} + \text{with my dog}_{\text{mu}}$. (私が**私の犬と**一緒に行った。) **→** 行為の主体（能格）とは別に、**同伴者**として付加的な役割を果たす。 |
| | **受動態** ($\text{l}$) | $\text{The car}_{\text{n}} + \text{was destroyed}_{\text{l}} + \text{with the garage}_{\text{mu}}$. (車が**ガレージと**一緒に破壊された。) **→** 受動態でも、破壊という動作に**同時に存在した付帯物**として機能。 |
| **3. 欠格** ($\text{w}\text{o}$) | **中動態** ($\text{n}$) | $\text{The theory}_{\text{n}} + \text{stands}_{\text{n}} + \text{without evidence}_{\text{wo}}$. (その理論は**証拠なしで**成り立っている。) **→** 中動態（自己影響）と結合し、**欠如**という状態の中で動作が完結することを表現。 |
| **4. 様格** ($\text{ʂ}\text{u}$) | **反射態** ($\text{y}$) | $\text{She}_{\text{pha}} + \text{dressed}_{\text{y}} + \text{with grace}_{\text{ʂu}}$. (彼女は**優雅な様子で**服を着た。) **→** 動作（反射態）の**質的な様態**を記述。手段格とは異なり、行為の性質に焦点を当てる。 |
| **5. 同格** ($\text{z}\text{i}$) | **使役受動態** ($\text{s}$) | $\text{He}_{\text{n}} + \text{was made to look}_{\text{s}} + \text{like a fool}_{\text{zi}}$. (彼は**愚か者のように**見させられた。) **→** 使役受動態（強制）と結合し、**類似性の基準**が強制の結果として設定される。 |
| **6. 限定格** ($\text{d}'\text{o}$) | **能動態** ($\text{∅}/\text{m}$) | $\text{We}_{\text{pha}} + \text{will only talk}_{\text{m}} + \text{as far as this chapter}_{\text{d'o}}$. (私たちは**この章に限定して**話す。) **→** 概念的な**限定範囲**を設定し、動作の適用範囲を制限する。 |
| **7. 観点格** ($\text{b}\text{i}$) | **全ての態** ($\text{m}, \text{n}, \dots$) | $\text{From my perspective}_{\text{bi}}, + \text{the car}_{\text{n}} + \text{was driven}_{\text{l}} + \text{by the wind}_{\text{tʃhy}}$. (**私の観点から**、車は風によって運転された。) **→** 文全体の解釈に**認知的な枠組み**を提供し、動詞の態や他の格の解釈を左右する。 |

意味論格の7個は、文の核から離れ、**動作や文脈の付加情報**（様態、欠如、同伴、観点）を表現する役割を、動詞の複雑な態（反射態、中動態、使役態）と独立して、または補完的に果たす。
