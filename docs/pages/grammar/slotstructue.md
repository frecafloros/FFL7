# 形態論 (Morphology)

## スロット構造

プリャンナの基本語は下記の構造を取る。

[語根] - [意味派生] - [語彙アスペクト] - [文法アスペクト] - [態] - [時制] - [法] - [数] - [格変化] - [証拠性] - [極性] - [語調]

|スロット|日本語名称|音素構造|形態論上の役割|
|:----|:----|:----|:----|
|1|[semantic derivation] (意味派生)|(CV)∗|語彙の拡張、意味の派生|
|2|[root] (語根)|CVC|動作の基本的な意味|
|3|[lexical aspect] (語彙アスペクト)|VC|動作の固有のタイプ（7種）|
|4|[grammatical aspect] (文法アスペクト)|yV|話者の時間的・様態的視点（8種）|
|5|[voice] (態)|C|行為者と被行為者の関係性（8種）|
|6|[tense] (時制)|wV|動作の時間軸上の位置|
|7|[mood] (法)|C|話し手の行為への態度|
|8|[grammatical number] (数)|V|対象の様相、まとまり方（7種）|
|9|[case] (格変化)|CV|名詞句の統語的・意味的役割（58種）|
|10|[evidentiality] (証拠性)|C|情報源の確実性|
|11|[polarity] (極性)|V|肯定/否定|
|12|[pragmatic] (語調)|CV|驚き、強調|

### 動名詞トークン

動名詞は、動詞として振る舞いながら名詞の位置（主語、目的語）に入る形態である。

- 動詞機能（4, 5, 6, 7: 行為者数、態、時制、法）を持つことで、動作性を維持する。
- 名詞機能（8, 9: 数、格）を持つことで、文中で名詞句として機能する。

### 動詞核トークン

動詞核は、非常に特殊で、従属節の「核」を形成することができる。

|スロット|理由|機能|
|--|--|--|
|1-7|動詞核|動作、時制、様相を表現。|
|8-9|名詞機能|従属節全体を名詞句として格付加（原因、時間など）するために使用。|
|10-12|文機能|従属節内の命題の情報源、真理値、感情を付加。|

例：「（君が、その事実を目撃したこと[1-9]）は、間違いない[11]と、私は確信を持って伝える[10]！」（感情を込めて[12]）
