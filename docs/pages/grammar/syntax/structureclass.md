# 機能語

機能語は、情報の関係性や構造そのものを記述するための語彙である。

- `:` → 「〜の内容は以下の通りである」 (定義、内容表示)
- `->` → 「〜から以下が導かれる」 (導出、因果、結果)
- `<->` → 「〜が以下と関連する」 (関連、対応、相互関係)
- `+` → 「〜は以下と組み合わさる」 (組み合わせ、合成、包含)

## 内容表示（-）

>     
> 
> *ahaa ṭa haa, taa, paa.*
>
> 人間というのは私とかあなたとか彼とかいうのである。

## 論理展開（-）

>   
>
> *eyaa ṣa wit'aa.*
> 
> *存在から善が導かれる。*

## 相互関係（-）

>    
>
> *ahaa ṇha wit'aa.*
> 
> *人間は善と関連する。*

## 累加合成（-）

>    
>
> *wit'aa č̣a d'üp!aa.*
> 
> *善と悪。*
