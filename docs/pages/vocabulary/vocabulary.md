# 語彙 (Vocabulary)

プリャンナの語彙は、その機能と音素構造に基づき、以下の二種類の語根に分類される。

## 1. 内容語根 (Content Roots)

具体的な概念、事物、動作、状態といった**語彙的な意味**を担う語根。

-   **音素構造:** 全て **CVC** (子音-母音-子音) の形式を取る。
-   **例:** `tan` (水), `tac` (飲む), `ṭhah` (知る) など。

## 2. 機能語根 (Functional Roots)

それ自体が具体的な語彙的意味を持つのではなく、主に**文法的な機能**を担う語根。

-   **音素構造:** 全て **C** (単一の子音) の形式を取る。
-   **例:** `h` (私), `j` (関係詞), `k'` (不定項) など。
