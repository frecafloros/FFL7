# 語用例: 「根本的な認識の転換」

- 基底状態（接辞なし）: √žel
  - 意味: 「根本的な認識の転換が存在している／転換された状態である。」
  - 例文: 「その教義は、すでに根本的な認識の転換（コペルニクス的転回）を遂げた状態である。」（客観的な事実として、その概念がもう新しい視点に変わっている。）
- 達成活動 (-iny): √žel-iny
  - 意味: 「（根本的な認識の転換を）達成する／見出す。」
  - 例文: 「彼は長い思索の末、ついに根本的な認識の転換を達成した。」（行動の結果として、その転換に到達した。）
- 使役能動態 (-k'): √žel-ak'
  - 意味: 「（他者に）根本的な認識の転換を促す／させる。」
  - 例文: 「その書物は、読者にコペルニクス的転回を促した。」（書物が読者の認識を変える原因となった。）
