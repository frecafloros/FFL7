# FFL7

FFL7の文法などを記述します。

## ディレクトリ構成

```
my-project/
├── .gitignore           # Git 管理に含めたくないファイルを記述
├── README.md            # 概要
├── docs/                # GitHub Pages の公開対象フォルダ（またはルートを使用する場合もあり）
│   ├── index.md         # サイトのトップページ（必要に応じて HTML に自動変換）
│   ├── about.md         # その他ページ（言語仕様や使い方など）
│   ├── assets/          # サイトに使用するスタイルシート、スクリプト、画像など
│   │   ├── css/
│   │   │   └── style.css   # カスタム CSS（自作フォントの定義なども含む）
│   │   ├── js/
│   │   │   └── main.js     # 追加のスクリプト（Markdown のレンダリング用など）
│   │   └── images/
│   │       └── logo.png    # 画像素材（ロゴ、背景など）
│   └── data/            # プロジェクト固有の設定ファイルやデータ（必要に応じて）
```
